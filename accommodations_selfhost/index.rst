Installation instructions for Roomify for Accommodations - Self-Hosted
======================================================================
Installation instructions for Roomify for Accommodations - Self-Hosted


Installing a self-hosted Roomify for Accommodations site is very simple.

1. Unzip file
2. Configure web server to point at directory
3. Import the included .sql file
4. Edit the existing sites/default/settings.php file and adjust database settings
5. Set appropriate permissions for the files directory (`Here <https://www.drupal.org/node/244924/>`_ is an excellent guide.)
6. Ensure php memory limit is at least 160M


NB: User1 user/pass is admin/admin
