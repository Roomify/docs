Rooms Documentation
==========================================
This documentation refers to the `Rooms 1.9 module`_.

.. _Rooms 1.9 module: https://drupal.org/project/rooms

Rooms 1.9 is a set of Drupal modules designed to enable individual holiday homes, B&Bs and hotels to define what is available for booking (whether it is the entire building or rooms within a building), manage the availability and pricing and integrate with Drupal Commerce to handle payment.
